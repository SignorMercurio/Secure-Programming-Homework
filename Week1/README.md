# Week 1

Generate random 120-bit prime with Miller-Rabin algorithm.

## Dependencies
- GNU MP Library

Tested under Ubuntu 16.04 & 19.04.

Note: `libgmp` is pre-installed on both of my VMs. In case of error, `gmpxx.h` and `libgmp.so.10` are provided.
